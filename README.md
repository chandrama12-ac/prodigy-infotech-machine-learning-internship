Projects
1. Iris Species Classification
Dataset: Iris Dataset
Algorithms: Logistic Regression, SVM, Random Forest, KNN
Accuracy: 97.8% (Random Forest)
Notebook: iris_classification.ipynb
2. House Price Prediction
Dataset: Boston Housing / Kaggle House Prices
Techniques: Feature Engineering, Polynomial Regression, XGBoost
Metrics: RMSE: 2.45, R²: 0.89
Notebook: house_price_prediction.ipynb
3. Customer Segmentation
Dataset: Mall Customer Segmentation Data
Algorithms: K-Means Clustering, Hierarchical Clustering
Visualization: Elbow Method, Dendrogram
Notebook: customer_segmentation.ipynb
4. Digit Recognition (MNIST)
Dataset: MNIST Dataset
Model: CNN with TensorFlow/Keras
Accuracy: 99.2% on test set
Notebook: mnist_cnn.ipynb
5. Sentiment Analysis
Dataset: IMDB Movie Reviews / Twitter Sentiment
Techniques: TF-IDF, LSTM, BERT (optional)
Accuracy: 87.5%
Notebook: sentiment_analysis.ipynb
6. Facial Emotion Recognition
Dataset: FER-2013
Model: CNN with Data Augmentation
Accuracy: 68.4%
Notebook: emotion_recognition.ipynb
